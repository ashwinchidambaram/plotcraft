Design a complete visual design system for a diagram/flowchart engine called PlotCraft. The style should feel hand-drawn and sketchy — think       Excalidraw, Balsamiq, or napkin sketches brought to life. Imperfect lines, slightly wobbly shapes, warm and approachable. It should look like a    smart person's whiteboard, not a corporate slide deck.                                                                                                                                                                                                                                                1. Color Palette (9 semantic themes)                                Design fill + stroke color pairs for each semantic category. Colors should feel like highlighter markers and felt-tip pens on paper — slightly     muted, warm, and organic:                                                                                                                                                      - Neutral — pencil-on-paper default (warm gray, not cold)                                                                                          - Start — "go" energy (soft green, like a green highlighter)   - End — completion/stop (warm rose or muted red)                                                                                                   - Decision — attention, branching (yellow highlighter)                                                                                             - Error — something went wrong (salmon/coral, not aggressive red)                                                                                  - Highlight — look here (soft blue/lavender highlighter)                                                                                           - Info — informational (teal/mint marker)                                                                                                          - Success — it worked! (warm green, distinct from Start)                                                                                           - Warning — heads up (peach/orange marker)                                                                                                                                                                                                                                                            For each: show a hand-drawn rectangle with slightly wobbly rounded corners, imperfect stroke (1.5-2px, slightly rough), and sample text inside     in a handwriting or monospace font. Fills should look like translucent highlighter over paper. Strokes should look like pen ink.                                                                                                                                                                      2. Shape Library (6 types)                                          Design each shape at roughly 180x90px with "Sample Text" inside, Neutral palette. Every shape should look hand-drawn — slightly imperfect edges,    like someone sketched it quickly but neatly:                                                                                                                                                      - Rectangle — rounded but wobbly corners                           - Square — not perfectly square, slightly organic   - Circle — hand-drawn circle (slightly imperfect)   - Oval/Ellipse — loose, organic shape                                                                                                              - Diamond — hand-drawn rotated square, text centered   - None (free-floating text) — just handwritten-style text, no shape                                                                                                                                                   Show each shape once in Neutral, then show the Rectangle in all 9 color themes side by side.                                                                                                                          3. Typography Scale (4 levels)                                                                                                                                                                                        Use a font that feels hand-written or casual but still readable. Good options: Virgil (Excalidraw's font), Caveat, Indie Flower, Patrick Hand,     or a clean monospace like Source Code Pro. Define size, weight, and color for:                                                                                                                                                      - Title — big, bold, like a whiteboard header written with a thick marker                                                                          - Subtitle — medium, like a section label   - Body — standard node text, comfortable reading size                                                                                              - Caption — small annotations, like pencil notes in the margin                                                                                         Show each as free-floating text and inside a Neutral rectangle.                                                                                                                                                       4. Connectors & Arrows                                                                                                                                                                                                Design connector styles that look hand-drawn:    - Solid line — slightly wobbly, like a pen stroke (define width and color)                                                                         - Dashed line — hand-drawn dashes (uneven spacing is fine)   - Dotted line — hand-drawn dots                                                                                                                    - 3 line weights: Thin (light pencil), Normal (pen), Bold (thick marker)                                                                                                                                                      For arrowheads:                                                    - Design a hand-drawn filled triangle arrowhead — slightly imperfect, like quickly drawn                                                           - Show: forward arrow, bidirectional arrow, plain line (no arrow)                                                                                  - Show a connector with a small handwritten label beside the midpoint                                                                                                                                                      5. Section Grouping                                                                                                                                                                                                   Design section containers that look like someone drew a loose boundary around a group of elements:                                                                                                                                                                                                    - Large rounded rectangle with wobbly edges, very subtle fill (like light chalk or whiteboard marker background)                                   - Section label in top-left in a casual handwriting style          - Show 3 section color schemes side by side                                                                                                        - Should feel like "this area of the whiteboard" not a rigid container                                                                                                                                                      6. Reference Composition                                                                                                                                                                                              Create one complete example diagram: a 5-7 node flowchart with a title, two sections, a diamond decision, mixed connector styles, and 3+ color     themes. It should look like a really well-done whiteboard sketch — the kind you'd photograph and share with your team because it explains things    perfectly.                                                                                                                                                                                                           Design Constraints    - All lines slightly imperfect/wobbly (not computer-straight)                                                                                      - Warm off-white canvas (#fdf6e3 or similar paper/cream tone)   - Shapes should have generous padding — nothing cramped                                                                                            - Colors accessible but warm — no neon, no cold corporate blues                                                                                    - The overall feel: approachable, human, smart-casual — like your favorite teacher's whiteboard drawings
